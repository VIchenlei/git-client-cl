## 安装

### 安装 node

> 如果已安装，请跳过本步骤。

1. `nodejs.org` 下载 `nodejs` 安装文件；
1. 按照提示安装 `nodejs`；
1. 命令行下执行 `node -v` 验证是否安装正确；

### 安装 client

1. `mkdir yaloc && cd yaloc`，创建并进入项目目录；
1. `git clone http://xxx/webapp/client.git .`，下载代码库到本地；
1. `npm install`, 安装相关的依赖；
1. `npm start`, 启动开发模式；
